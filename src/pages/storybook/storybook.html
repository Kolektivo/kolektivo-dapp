Icons To keep the bundle size to a minimum, we only import the icons we need. To import a new icon:
  1. Go to https://fonts.google.com/icons and find the icon name you need
  2. Open app-container.ts
  3. Import the icon from the @material-design-icons package (you can import the icon from whichever folder you need [filled, outlined, round, sharp, two-tone])
  4. Register that icon in the DesignSystemPlugin
  5. Add the newly imported icon to this section so we can all see :)
